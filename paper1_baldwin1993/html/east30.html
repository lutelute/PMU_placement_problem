IEEJ EAST 30機系統について

電気学会（IEEJ）が策定した東日本の送電系統標準モデル。関東・東北を模した50Hz系統です。

系統規模（昼間条件）: 93,880 MVA (73,540 MW) / 負荷 72,600 MW
データ: 公式Y法形式ファイルからパース (east30p/s.txt)

ノード表示ルール

ノードの色 = 電圧階級（常時表示）
・赤丸 = 500kV ・橙丸 = 275kV ・緑丸 = 154kV ・黄丸 = 発電機

PMU状態 = リング・グローで重畳表示
・白太枠 + 赤グロー = PMU設置済み
・緑枠 = PMUなしだが観測済み
・枠なし（暗い色） = 未観測

操作方法

  • ノードをクリック → PMU設置/撤去
  • 「最適配置」ボタン → Greedy / SA で自動求解
  • マウスホイール → ズーム
  • 右ドラッグ or Shift+ドラッグ → パン
  • ノードにホバー → 詳細ツールチップ
