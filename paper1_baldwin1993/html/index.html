Paper 1: PMU Optimal Placement for Power System Observability

T.L. Baldwin, L. Mili, M.B. Boisen Jr., R. Adapa (1993) — IEEE Trans. Power Systems

1. 論文の背景と目的

PMU（Phasor Measurement Unit）は電力系統のバスにおいて電圧フェーザーを直接測定し、 接続する送電線の電流フェーザーも同時に測定できる装置です。 本論文は完全観測性（Complete Observability）を達成するために 必要なPMUの最小数と最適配置を求める問題を初めて体系的に定式化しました。

2. PMU観測性ルール

Rule 1: PMUが設置されたバスは直接観測可能
Rule 2: PMUが設置されたバスに隣接する全てのバスも観測可能 （PMUは設置バスの電圧と全ての接続ブランチの電流を測定するため、 オームの法則 Vj = Vi - Zij·Iij から隣接バスの電圧が計算可能）

3. 数学的定式化（BILP）

接続行列 f を定義します：

fij = 1 if i = j（自分自身）or バスiとバスjが直接接続
fij = 0 otherwise

PMU配置ベクトル x：

xi = 1 if バスiにPMUを設置
xi = 0 otherwise

最適化問題：

最小化: Σ xi (i = 1, ..., N)
制約条件: f · x ≥ 1 (全てのバスが少なくとも1つのPMUで観測される)
変数: xi ∈ {0, 1}

4. 解法：二分探索 + シミュレーテッドアニーリング

Baldwin et al. は以下のハイブリッド手法を提案しました：

  • 二分探索： PMU数 k を二分探索で探索
  • SA： 各 k について、k個のPMUで完全観測性が達成可能かSAで判定

SAの手順

  • 初期解：k個のバスをランダムに選択してPMUを配置
  • 近傍操作：PMU設置バスと非設置バスを1組スワップ
  • 評価関数：未観測バス数（= 0 なら実行可能解）
  • 冷却スケジュール：T(t+1) = α · T(t), α ≈ 0.995

6. 観測深度（Observability Depth）
