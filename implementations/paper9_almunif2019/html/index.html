Paper 9: MILP vs NLP によるPMU最適配置の比較

A. Almunif, L. Fan (2019) — North American Power Symposium (NAPS)

DOI: 10.1109/NAPS46351.2019.9000371

1. 論文の背景と目的

PMU最適配置問題は通常、混合整数線形計画法（MILP）で定式化されますが、 本論文は非線形計画法（NLP）による代替定式化を提案し、 MILPとの結果を比較しています。NLPアプローチは整数変数を連続緩和し、 異なる初期点から複数の最適解を発見できるという利点があります。

2. MILP定式化

Minimize: ∑i=1n xi
Subject to: A · x ≥ 1
xi ∈ {0, 1} ∀i

A: 接続行列 (aij = 1 if bus i,j connected or i=j)

3. NLP定式化

Minimize: ∑i=1n xi
Subject to: A · x ≥ 1
0 ≤ xi ≤ 1 ∀i
xi(1 - xi) ≤ 0 ∀i (整数性を強制する非線形制約)

NLPでは整数制約を非線形制約 xi(1-xi) ≤ 0 で置き換えます。 異なる初期点から解くことで、複数の最適解を発見できます。

4. 拡張シナリオ

ゼロインジェクションバス（ZIB）の考慮

ZIBとは発電も負荷もないバスで、KCL（キルヒホフの電流法則）により 追加の観測性情報が得られます。IEEE 14-busではバス7がZIBです。 ZIBを考慮すると、必要PMU数が4から3に削減されます。

PMU故障への冗長性

各バスが少なくとも2台のPMUで観測可能（N-1冗長性）を要求すると、 必要PMU数は大幅に増加します。

チャネル数制限

PMUが同時に測定できるチャネル数に上限 c を設けると、 高接続度バスへのPMU配置の効果が制限されます。
