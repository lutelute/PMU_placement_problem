Paper 5: Branch PMU Placement Model

R. Emami, A. Abur (2010) — IEEE Trans. Power Systems, Vol. 25, No. 2

DOI: 10.1109/TPWRS.2009.2036474

1. 論文の背景と目的

従来のPMU配置モデルでは、PMUはバスに設置され、 そのバスの電圧フェーザーと全ての接続ブランチの電流フェーザーを測定するモデルでした。 Emami & Abur (2010) は、PMUをブランチ（送電線）に設置する 新しいモデルを提案しました。

2. ブランチPMUモデル

ブランチPMU： 送電線(i,j)に設置されたPMUは、 その線路の電流フェーザー Iij と、 両端のバスの電圧フェーザー Vi, Vj を測定します。

ブランチPMUの観測ルール：

  • Rule 1: ブランチPMUが設置された送電線の両端バスが直接観測される
  • Rule 2: あるバスの電圧が既知で、接続するブランチの電流が既知なら、 オームの法則により対向バスの電圧が計算可能

3. 数学的定式化

決定変数はブランチごとに定義されます：

yij = 1 if ブランチ(i,j)にPMUを設置
yij = 0 otherwise

バス i の観測条件：

最小化: Σ yij (全ブランチ(i,j)について)
制約条件: 各バス i について、i を端点とするブランチの少なくとも1つにPMUが設置
Σj:(i,j)∈E yij ≥ 1 ∀ i ∈ V
変数: yij ∈ {0, 1}

4. バスPMUとの比較

バスPMU： 1台で設置バス+全隣接バスを観測。チャネル数 = 接続ブランチ数。コスト高。
ブランチPMU： 1台で両端バスのみ観測。1チャネル。コスト低。ただし台数は増加。

ブランチPMUは個々のコストが安いため、台数が増えても総コストが削減される場合があります。 また、ブランチPMUはCT（変流器）の近くに設置できるため、設置が容易な場合があります。
